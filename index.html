COVID-19 DATA EXPLORATION & IMPACT ANALYSIS

This project is focused on analyzing COVID-19 data using SQL to uncover critical insights about the global spread, death rates, and vaccination progress. Key tasks included:

  • Case vs Death Analysis: Examined the ratio of deaths to cases across regions to understand the severity of the pandemic.
  • Infection Impact on Population: Calculated the percentage of populations affected by COVID-19 in various locations.
  • Vaccination Progress: Compared vaccination rates to total cases, tracking the global vaccination effort.
  • Advanced SQL Techniques: Utilized CTEs, temporary tables, and views to efficiently manipulate and store data for future analysis.

This project demonstrates a comprehensive view of COVID-19’s impact on a global scale.

MOVIE CORRELATION WITH PYTHON

This project explores the relationship between movie budgets, earnings, and influential factors. Key tasks included:

  • Budget vs Gross Earnings: Created a scatter plot to test the hypothesis, revealing a strong correlation between budget and gross earnings, validating the hypothesis.
  • Correlation Analysis: Converted object values to numeric and ran a correlation matrix, showing that votes and budget had the highest correlation to gross earnings.
